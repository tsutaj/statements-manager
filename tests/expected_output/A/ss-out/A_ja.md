


## Problem A

### <u>**A+B**</u>

2 つの整数 $A$, $B$ が与えられます。$A + B$ の計算結果を出力してください。

### 入力

入力は以下の形式で標準入力から与えられます。

```
$A$ $B$
```

### 出力

$A + B$ の計算結果を標準出力に出力してください。

### 制約

- $1 \leq A, B \leq 10^{9}$



### 入力例 1
```
3 5
```



### 出力例 1
```
8
```





### 入力例 2
```
10000 100
```



### 出力例 2
```
10100
```
