Problem C2

Static Range Sum

問題文

長さ $N$ の非負整数列 $A = (a_0, a_1, \ldots, a_{N-1})$ が与えられます。以下で説明されるクエリを順に $Q$ 回処理してください。

  • 整数 $l_i$​ と $r_i$​ が与えられるので、$\sum_{k=l_i}^{r_i-1} a_k$​ を出力する。

注意

問題 C1 と C2 では、$N$ と $Q$ の制約のみが異なります。

入力

$N$ $Q$
$a_0$​ $a_1$​ $\ldots$ $a_{N-1}$​
$l_1$​ $r_1$​
$\vdots$
$l_Q$​ $r_Q$​
  • 入力は全て整数で与えられる
  • $1 \leq N \leq 100{,}000$
  • $1 \leq Q \leq 100{,}000$
  • $-10^{9} \leq a_i \leq 10^{9}$
  • $1 \leq l_i < r_i \leq N$

入出力例

入力例

5 5
1 10 100 1000 10000
2 3
0 3
2 5
3 4
0 5

出力例

100
111
11100

11111
